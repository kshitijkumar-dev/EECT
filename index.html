DC Circuits & Machines

KCL · KVL · DC Motors · Tariffs · Stepper Motor
Q1
Differentiate between Active and Passive Elements with suitable examples.+
Definition

Active Elements

Components that supply energy to a circuit and can provide power gain.

  • Battery
  • Generator
  • Solar Cell
  • Transistor

Passive Elements

Components that cannot generate energy — they only store, dissipate, or absorb it.

  • Resistor
  • Capacitor
  • Inductor
  • Transformer
Active ElementsPassive Elements
Supply energyConsume / store energy
Provide gainNo gain
Require external sourceNo external source needed
Battery, GeneratorResistor, Capacitor
Q2
Define Electrical Load. Explain types of electrical loads.+
Definition

Electrical Load: Any device that consumes electrical energy and converts it into heat, light, or mechanical energy.

Types

  • Resistive Load — Current & voltage are in phase. Ex: Heater, Iron box, Incandescent lamp.
  • Inductive Load — Current lags voltage. Ex: Motor, Transformer, Fan.
  • Capacitive Load — Current leads voltage. Ex: Capacitor bank, Underground cables.
Q3
State and Explain Kirchhoff's Current Law (KCL).+
Definition

Statement: The algebraic sum of all currents at a node is zero.

∑I = 0    OR    I_in = I_out

Example

Current entering = 10 A, one branch = 6 A, find other branch:

10 = 6 + I  →  I = 4 A ✓
Q4
State and Explain Kirchhoff's Voltage Law (KVL).+
Definition

Statement: The algebraic sum of all voltages around a closed loop is zero.

∑V = 0

Example

Supply = 24V, V₁ = 10V, V₂ = 14V

24 − 10 − 14 = 0 ✓

Q7
What are Electrical Tariffs? Explain different types.+

Tariff: The rate charged to consumers for electrical energy consumed.

Types

  • Flat Rate — Fixed charge per unit
  • Block Rate — Different rates for different consumption slabs
  • Maximum Demand — Based on peak demand drawn
  • Time-of-Day — Peak vs off-peak pricing
  • Two-Part — Fixed charge + energy charge
  • Seasonal — Different rates across seasons

Describe Working Principle of DC Machine.+

Works on electromagnetic interaction.

  • As Generator: Mechanical Energy → Electrical Energy (Faraday's Law)
  • As Motor: Electrical Energy → Mechanical Energy (current-carrying conductor in magnetic field experiences force)
Q11
Explain Working of a Stepper Motor.+

Definition: Converts electrical pulses into discrete angular movements.

Working Steps

  1. Stator winding energized
  2. Magnetic field produced
  3. Rotor aligns with field
  4. Next pulse energizes another winding
  5. Rotor moves one step
  6. Continuous pulses → rotation

Applications

CNC machines, Robotics, Printers, 3D printers

Q1
Explain Phasor Representation of Sinusoidal Quantities.+

Phasor: A rotating vector representing magnitude and phase angle of a sinusoidal quantity.

v(t) = Vₘ sin(ωt + φ) Phasor form: V = V∠φ

Advantages

  • Simplifies AC circuit analysis
  • Easy addition/subtraction of sinusoidal quantities
  • Clearly shows phase difference

Q6
What is Peak Value of an AC waveform?+

Peak Value (Vₘ or Iₘ): The maximum value attained by voltage or current during one complete cycle.

Q7
What is RMS Value of an AC waveform?+

RMS Value: Equivalent DC value that produces the same heating effect.

V_rms = Vₘ / √2 = 0.707 × Vₘ I_rms = 0.707 × Iₘ
Q8
What is Instantaneous Value of an AC waveform?+

Instantaneous Value: Value of voltage/current at any particular instant of time.

v(t) = Vₘ sin(ωt)
Q9

Power Factor (PF) = cos φ = P / S

Significance

  • Indicates efficiency of power usage
  • Reduces transmission losses
  • Improves voltage regulation
  • Lowers electricity cost
  • Increases system capacity
Q10
Explain Construction of Three-Phase Induction Motor.+
  • Stator — Stationary part; contains three-phase winding
  • Rotor — Rotating part; squirrel cage or wound type
  • Air Gap — Space between stator and rotor
  • Shaft — Delivers mechanical output power
  • Bearings — Support rotor rotation
Q11
Describe Working Principle of Three-Phase Induction Motor.+

Principle: Electromagnetic Induction

  1. Three-phase supply applied to stator
  2. Rotating magnetic field (RMF) produced
  3. Rotor conductors cut the magnetic flux
  4. EMF induced in rotor (Faraday's Law)
  5. Rotor current flows
  6. Torque developed → rotor rotates
Q12
Explain Working of a Servomotor.+

Definition: A motor used for precise position and speed control.

  1. Input command applied
  2. Error detector compares actual vs desired position
  3. Controller sends correction signal
  4. Motor rotates to correct position
  5. Feedback continuously corrects error

Applications

Robotics, CNC machines, Automation, Aerospace

Diode · BJT · MOSFET · Open/Closed Loop
Q1
Define a PN Junction Diode.+
Definition

A PN junction diode is a two-terminal semiconductor device formed by joining P-type and N-type semiconductor materials.

  • Allows current in forward direction
  • Blocks current in reverse direction
  • Terminals: Anode (P-side), Cathode (N-side)
Q2
Name the three terminals of a BJT (Bipolar Junction Transistor).+
  • Emitter (E) — Emits charge carriers
  • Base (B) — Controls transistor operation
  • Collector (C) — Collects charge carriers
Q3
What are the terminals of a MOSFET?+
  • Gate (G) — Controls channel conductivity
  • Drain (D) — Current exits here
  • Source (S) — Current enters here
  • Body/Substrate — (in some MOSFETs)

Q6
What is Forward Bias in a Diode?+

Forward Bias: Positive terminal connected to P-side, Negative terminal to N-side.

Effects

  • Depletion layer decreases
  • Barrier potential reduces
  • Current flows easily
  • Diode behaves like a closed switch
Q7
Explain the Working Principle of PN Junction Diode.+

Forward Bias

+ve → P-side, −ve → N-side → Depletion narrows → Current flows

Reverse Bias

+ve → N-side, −ve → P-side → Depletion widens → Current blocked

Applications

Rectification, signal processing, protection circuits

Q9
A robotic system produces large oscillations for small input. Is it BIBO stable?+

BIBO: Bounded Input → Bounded Output

A system is BIBO stable if every bounded input produces a bounded output.

Analysis

  • Input is bounded (small)
  • Output is excessively large (unbounded)

Conclusion: The system is NOT BIBO stable. ❌

Signals & Digital Systems

CT/DT · Even/Odd · Periodic · Analog/Digital
Q1
Define a Continuous-Time (CT) Signal.+

A signal defined at every instant of time.

Examples: Temperature variation, ECG signal, Speech signal, AC waveform.

Q2
What is a Discrete-Time (DT) Signal?+

A signal defined only at discrete time intervals.

Examples: Digital meter readings, Computer data, Sampled signals.

Q3
Define an Even Signal.+

A signal is even if it is symmetric about the vertical axis.

x(t) = x(−t)

Example: x(t) = t²

Q4
Define an Odd Signal.+

A signal is odd if it is antisymmetric about the vertical axis.

x(t) = −x(−t)

Example: x(t) = t

Q5
What is a Periodic Signal?+

A signal that repeats after a fixed time period T.

x(t) = x(t + T)

Example: Sinusoidal AC waveform

Q6
Define an Aperiodic Signal.+

A signal that does not repeat itself over time.

Examples: ECG signal, Speech signal, Random noise

Q7
What is an Energy Signal?+

A signal with finite energy and zero average power.

0 < E < ∞    and    P = 0

Example: Finite duration pulse

Q8
Define a Power Signal.+

A signal with finite average power but infinite energy.

0 < P < ∞    and    E = ∞

Example: Periodic sinusoidal signal
